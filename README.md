# CSV Localizator

CSV Localizator is an Unreal Engine editor plugin for converting Unreal localization `.po` files to editable `.csv` files and importing edited CSV files back into `.po`.

It is useful if you prefer editing translations in Google Sheets, Excel, AI translation tools, or any workflow where CSV is more convenient than Unreal's PO format.

## Features

- Export Unreal `.po` localization files to `.csv`
- Import edited `.csv` files back to `.po`
- Export all cultures into one CSV file
- Export native culture + one selected culture
- Automatically detects localization targets and cultures from the project
- Creates `.po` backups before overwriting files
- Runs inside the Unreal Editor
- Uses Unreal's built-in Python
- No system Python installation required

## Requirements

- Unreal Engine 5.x
- Python Editor Script Plugin enabled
- Editor Scripting Utilities enabled
- C++ build tools if using the source version of the plugin

This plugin contains a C++ editor module.  
Blueprint-only projects can still use it, but the source version requires Unreal C++ build tools to compile the plugin.

On Windows, install:

- Visual Studio 2022 Build Tools
- MSVC toolchain
- Windows SDK

If you use a precompiled release for your exact Unreal Engine version, no IDE is required.

## Installation

### Option 1: Precompiled Release ZIP

Download the release ZIP matching your Unreal Engine version from the GitHub Releases page:

```
CsvLocalizator-v1.0-UE5.6-Win64.zip
CsvLocalizator-v1.0-UE5.7-Win64.zip
```

Extract it into your project:

```text
YourProject/
  Plugins/
    CsvLocalizator/
      CsvLocalizator.uplugin
      Source/
      Content/
      Resources/
```

### Option 2: Source Version

Use this option if you want to modify the plugin or build it for another Unreal Engine version.  
Use the source archive that includes the plugin. Place it in the Plugins folder. Then regenerate project files and build your project editor target.



## Unreal Localization Workflow

CSV Localizator works with .po files generated by Unreal's Localization Dashboard.

Recommended workflow:

1. Open Localization Dashboard
2. Configure your localization target
3. Gather Text
4. Export Text
5. Open CSV Localizator
6. Export PO to CSV
7. Edit CSV externally
8. Import CSV to PO
9. Import/Compile localization in Unreal

The plugin expects Unreal's standard localization layout:
```
Content/Localization/<Target>/<Culture>/<Target>.po
```
Example:
```
Content/Localization/Game/en/Game.po
Content/Localization/Game/ru/Game.po
Content/Localization/Game/de/Game.po
```


## Python Dependency

CSV Localizator uses polib for reading and writing .po files.  
The plugin uses Unreal's built-in Python.  
System Python is not required.  
polib is bundled with the plugin, so users do not need to install Python packages manually.



## No PO files found

Make sure you ran:
```
Localization Dashboard -> Gather Text
Localization Dashboard -> Export Text
```
The plugin needs .po files to exist before exporting CSV.


## Notes

CSV Localizator does not replace Unreal's Localization Dashboard.  
It only adds a convenient bridge:
```
PO <-> CSV
```
Use Unreal's built-in localization system for gathering, importing, and compiling localization data.
